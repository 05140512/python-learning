## 二、主线:全栈地基（重心)

### A. 数据库 / SQL —— 打到"任意数据库都能上手"

目标不是背某个库的八股,而是掌握**所有关系型数据库共通的底层模型**,这样换 MySQL / PostgreSQL / SQLite 都能迁移。

> ⚠️ 关键:索引 / 优化的深入概念(联合索引、覆盖索引、回表、最左匹配)在 **SQLite 上看不到效果**(引擎不同)。这些要在 **PostgreSQL 或 MySQL** 上练,知识才真正落地、可迁移。建议本地装一个 PostgreSQL 作为"深入练习库",SQLite 继续作为项目轻量库。

学习顺序:

```txt
1. EXPLAIN / 执行计划         —— 先在 SQLite 入门,再到 PostgreSQL 看真实计划
2. 索引原理（B-tree 为什么快） —— 通用,跨库适用
3. 主键 / 唯一 / 普通索引
4. 联合索引 / 覆盖索引 / 最左匹配 —— ⚠️ 在 PostgreSQL/MySQL 上练
5. SQLAlchemy 中创建与使用索引
6. 真实 SQL 优化案例（慢查询 → 加索引 → 对比执行计划）
7. 事务与隔离级别（读未提交/已提交/可重复读/串行化）—— 跨库核心
8. 锁（行锁/表锁/死锁）         —— 跨库核心
9. 范式与反范式 / 数据建模      —— 全栈设计能力
```

### B. 数据库工程化


```txt

```

### D. 部署 / 工程化（主线收尾,初中级够用,只学生产最常用)

> 放在主线最后:有了能跑的全栈应用,再学"怎么把它部署上线"。求职常考、生产必用。

**Docker（初中级）**

```txt
镜像 vs 容器概念
docker run / ps / logs / exec / stop / rm 常用命令
Dockerfile 写法（给 FastAPI 应用打镜像）
docker compose（一键起 FastAPI + PostgreSQL + Nginx)
数据卷 volume（数据库数据持久化)
不学:k8s / 集群 / 复杂编排
```

**Nginx（初中级）**

```txt
反向代理（Nginx → FastAPI/uvicorn)
静态资源托管（部署你的 Vue3 前端打包产物)
配置 location / server 块
HTTPS / 证书基础
负载均衡概念（了解即可)
不学:深度调优 / 复杂 rewrite
```

🎯 产出:把"全栈 + 工业监控"项目用 docker compose 一键拉起、Nginx 做入口,完整上线一次。
