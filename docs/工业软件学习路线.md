# 二、学习总路线（推荐顺序）

## 第一阶段：Python 入门（2~4周）

目标：

- 从前端思维切换到工具开发思维
- 建立 Python 熟练度

---

## 学习内容

### 1. Python 环境

学习：

- Python 安装
- pip
- venv 虚拟环境
- VS Code 配置

掌握命令：

```bash
python --version
pip --version
pip install requests
```

---

### 2. Python 基础语法

重点学习：

- 变量
- list
- dict
- 函数
- class
- 文件操作
- json
- try except
- async await

不要刷算法。

---

### 3. Python 小工具项目

建议项目：

- Excel 转 JSON
- 自动重命名文件
- 日志分析
- 自动生成日报
- 图片压缩
- PDF 导出

目标：

- 提升 Python 熟练度
- 建立项目感

## 工业协议

### 1. Modbus

重点：

- 寄存器
- TCP 通讯
- 读写设备数据

Python：

```bash

```

---

### 2. OPC UA

重点：

- 工业互联网标准协议
- 点位系统
- 实时工业数据

Python：

```bash
pip install opcua
```

---

### 3. MQTT

重点：

- 工业 IoT
- 实时消息

理解为：

- 工业版 WebSocket
